 Restaurant App

A responsive full-stack Restaurant web application built using the MERN stack. This app allows users to browse the menu, place orders, and make table reservations.

 Features

- User-friendly interface
- View restaurant menu
- Place online orders
- Book table reservations
- Full-stack integration using MERN

Tech Stack

- **Frontend**: React, Tailwind CSS
- **Backend**: Node.js, Express.js
- **Database**: MongoDB
- **Tools**: Git, VS Code
